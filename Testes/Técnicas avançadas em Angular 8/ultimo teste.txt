É possível reduzir o número de requisições que fazemos ao back-end no filtro do campo texto que inserimos na listagem de filmes?

O que é o RxJS?
O RxJS é uma biblioteca para programação reativa usando Observables.

Como saber que um campo está com erro?
Cada campo possui uma propriedade chamada erros e nela tem um array com todos os erros do campo que estão acontecendo.

Qual a função do JSON-Server?

Como podemos transformar uma data para o formato brasileiro?
O Angular possui um pipe pronto para isso, basta passar o pipe date e o formato desejado.
